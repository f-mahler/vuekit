# vuekit
Kirby and Vue together.

### Requirement
* A server with php, node, php (ex. Laravel Valet or MAMP)

### Project setup
1. Clone or copy repo
2. cd vuekit
3. npm install
4. Install kirby via /panel
5. Add username / password to /src/auth.js
6. npm run build

### Development
```
npm run dev
```

### Build for production
```
npm run build
```
